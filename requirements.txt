---------------------------------------------------------------
SOFTWARE REQUIREMENTS
---------------------------------------------------------------
  - Windows 10/11 (tested), Linux or macOS also supported
  - CMake 3.20+          https://cmake.org/download/
  - vcpkg                https://github.com/microsoft/vcpkg
  - C++17 compiler       (MSVC 2022, GCC 11, or Clang 14)
  - OpenGL 3.3+          (any GPU made after ~2010)

vcpkg packages required:
  glfw3, glad, glm
  Install with: .\vcpkg install glfw3 glad glm

---------------------------------------------------------------
BUILD INSTRUCTIONS
---------------------------------------------------------------
  Option A — Visual Studio 2022 (recommended):
    1. Open the ReceptionDesk folder in Visual Studio 2022
    2. It will detect CMakePresets.json automatically
    3. Select the "windows-default" preset if prompted
    4. Build -> Build All (Ctrl+Shift+B)

  Option B — Command line:
    cmake -B build -S . ^
      -DCMAKE_TOOLCHAIN_FILE="C:/Users/<you>/vcpkg/scripts/buildsystems/vcpkg.cmake"
    cmake --build build --config Debug

---------------------------------------------------------------
RUNNING THE PROGRAMS
---------------------------------------------------------------
  Scene.exe          — Project 5: Full lobby scene with Phong lighting
  ReceptionDesk.exe  — Lab: Reception counter with Phong lighting
  Pyramids.exe       — Lab: Three pyramids, three camera views
  Mesh.exe           — Lab: Sine wave surface mesh f(x,z)=sin(x)*cos(z)

  Controls (Scene.exe and ReceptionDesk.exe):
    W / S           — move camera forward / backward
    A / D           — strafe camera left / right
    LEFT / RIGHT    — rotate camera yaw
    UP / DOWN       — tilt camera pitch
    ESC             — quit

  Controls (Pyramids.exe):
    1 / 2 / 3       — switch between three camera views
    ESC             — quit

  Controls (Mesh.exe):
    LEFT / RIGHT    — rotate around Y axis
    UP / DOWN       — rotate around X axis
    ESC             — quit

---------------------------------------------------------------
LIGHTING (Project 5 / Scene.exe)
---------------------------------------------------------------
  Two point lights are used:
    Light 1: warm white (1.0, 0.97, 0.90) at position (0, 3.5, 0)
             simulates overhead recessed ceiling fixtures
    Light 2: cool blue fill (0.55, 0.60, 0.70) at position (-3, 2, 5)
             simulates ambient reflected light from exterior

  Per-object Phong material properties (specular strength + shininess)
  are tuned per surface type:
    Matte paint  — low spec (0.05-0.15), low shine (8-24)
    Laminate top — moderate spec (0.35), moderate shine (48)
    Glass/screen — high spec (0.70-0.80), high shine (128)
    Metal        — high spec (0.50-0.60), high shine (64-96)

---------------------------------------------------------------
TROUBLESHOOTING
---------------------------------------------------------------
  Black window / no output:
    Check that your GPU supports OpenGL 3.3.

  CMake cannot find glfw3/glad/glm:
    Run: .\vcpkg install glfw3 glad glm
    Ensure CMakePresets.json points to your vcpkg location.

  Scene looks flat / no lighting:
    Make sure you are running Scene.exe (Project 5), not the
    older build. Delete the out/ folder and rebuild if unsure.
